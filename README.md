# 🎬 Netflix Data Analysis using SQL

## 📌 Overview
This project analyzes Netflix data (8000+ titles) using PostgreSQL to uncover trends in content, ratings, genres, and regional distribution.

## 📊 Dataset Columns
- show_id
- type
- title
- director
- casts
- country
- date_added
- release_year
- rating
- duration
- listed_in
- description

## 🔍 Key Insights
- Movies dominate Netflix content (~70%)
- TV-MA is the most common rating
- USA & India are top content producers
- 2600+ missing director values (mostly TV Shows)
- 869 documentary movies available

## 🛠 Tools Used
- PostgreSQL
- pgAdmin
- Microsoft Excel

## 📁 Files Included
- Dataset
- SQL Queries
- Insights Report (PDF)
